================================================================================
  REQUIRED ENVIRONMENT VARIABLES
================================================================================

DATABASE_URL
  Type: String
  Format: postgresql://user:password@host:port/database
  Required: Yes
  Environments: All

JWT_SECRET
  Type: String
  Length: Min 32 characters
  Required: Yes
  Environments: Production only

JWT_REFRESH_SECRET
  Type: String
  Length: Min 32 characters
  Required: Yes
  Environments: Production only
  Note: Must be different from JWT_SECRET

NODE_ENV
  Type: String
  Value: production
  Required: Yes
  Environments: Production only

CORS_ORIGIN
  Type: String
  Example: https://medical-app-xyz.vercel.app
  Required: No (default: http://localhost:3000)
  Environments: Production only
